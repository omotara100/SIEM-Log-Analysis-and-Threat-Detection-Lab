# SIEM Log Analysis and Threat Detection Lab

## Overview

This project demonstrates how to deploy a SIEM and perform threat detection using centralized log analysis.

## Technologies Used

• Splunk SIEM  
• Sysmon  
• Windows Security Logs  
• PowerShell Logging  

## Skills Demonstrated

Log ingestion

Threat detection

Security event analysis

Detection engineering

SOC investigation workflow

## Investigations Performed

1. Suspicious PowerShell execution
2. Brute force login attempts
3. Network scanning activity

## Key Learning Outcomes

Centralized logging enables security analysts to detect and investigate malicious activity across systems.
